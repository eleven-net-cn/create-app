# Docsify Copy Code Plugin

docsify 的代码复制插件，一键复制代码块。

## 版本信息

- **来源**: docsify-copy-code@2.1.1
- **文件**: docsify-copy-code.min.js
- **大小**: 3.6KB
- **功能**: 代码复制、复制反馈

## 配置

```javascript
window.$docsify = {
  copyCode: {
    buttonText: '复制',      // 按钮文本
    errorText: '错误',       // 错误提示
    successText: '已复制'    // 成功提示
  }
}
```

## 特性

- 一键复制代码块
- 复制成功/失败反馈
- 可自定义按钮文本
- 支持所有代码语言
- 响应式设计
